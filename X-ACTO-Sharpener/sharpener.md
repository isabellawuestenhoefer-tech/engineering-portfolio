# X-ACTO Bulldog Vacuum Mount Manual Pencil Sharpener

## Overview
This project was a SolidWorks model of an X-ACTO pencil sharpener assembly.

## Skills Demonstrated
- Part modeling
- Assembly modeling
- Mates
- Design intent
- Fastener placement
- FEA Analysis

## Notes
This project showed  the difficulties that arise when creating a multi-part
assembly. I learned how to create spiral gears by hand and how to connect multiple 
gears together. I utilized a hand crank to spin the gear, allowing the pencil to be
sharpened. In addition, a vacuum seal was also created with a lever, allowing the 
pencil sharpener to be suctioned to the surface underneath it.
Difficulties arose throughout the project. Without a caliper, it was challenging to 
get measurements for internal parts, opening up the floor to estimating, testing if 
the measurement worked, then re-modeling if not. 
This project taught me resilience and the importance of tweaking dimensions and sketches.
In addition, I learned how parts can be trickier than they seem. Building a part, realizing 
it doesn't work, then trying a different approach to build it was a common theme.
